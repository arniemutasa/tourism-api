# TouriZim API

> Backend API for the TouriZim Application

## Usage

Rename "config/sample.config.env" file to "config/config.env" and update values to your own

## Install Dependancies

```
npm install
```

## Run App

```
# Run in Dev Mode
npm run dev

# Run in Production
npm start
```
